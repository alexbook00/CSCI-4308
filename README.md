# Run Flask App

1. Install Flask Requirements by running command `pip install requirements.`
 
2. Set App File and Run with following commands
 
(Mac/Linux)
`export FLASK_APP=app.py`

(Windows)
`set FLASK_APP=app.py`

`flask run`
